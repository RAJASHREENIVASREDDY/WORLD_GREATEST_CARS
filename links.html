Web Concept
HTML Links
Anchor tags — internal, external, email, anchor & country navigation
Type 1 — Internal Page Links

Links that navigate between pages within this website. Used in our navbar and all navigation buttons.

Internal Links
<!-- Navigate to another page in same project -->
<a href="germany.html">Germany 🇩🇪</a>
<a href="index.html">Home</a>
<a href="quiz.html">Take the Quiz</a>
Live Output
🏠 Home 🇩🇪 Germany 🇮🇹 Italy 🏁 Quiz 📊 Spec Table
Type 2 — External Links (open in new tab)

Links to websites outside this project. target="_blank" opens in a new tab.

External Links
<!-- target="_blank" opens in new tab -->
<a href="https://www.porsche.com" target="_blank">
  Porsche Official Site
</a>

<a href="https://www.ferrari.com" target="_blank">
  Ferrari Official Site
</a>
Live Output
🚗 Porsche Official ↗ 🏎️ Ferrari Official ↗ 👹 Nissan Official ↗ 🕷️ McLaren Official ↗
Type 3 — Email Links

mailto: opens the user's email client with the address pre-filled.

Email Link
<!-- Opens mail client with address filled in -->
<a href="mailto:contact@autocarta.com">
  Contact Us
</a>

<!-- With subject pre-filled -->
<a href="mailto:contact@autocarta.com?subject=Car Enquiry">
  Send Car Enquiry
</a>
Live Output
✉️ contact@autocarta.com 📧 Send Car Enquiry
Type 4 — Anchor Links (Jump to section)

Links to a specific section on the same page using an id.

Anchor Links
<!-- The destination anchor -->
<div id="germany-section">...</div>

<!-- The link that jumps to it -->
<a href="#germany-section">Jump to Germany</a>

<!-- Back to top -->
<a href="#top">↑ Back to Top</a>
Live Output
Jump to Germany ↓ Jump to Italy ↓ ↑ Back to Top
