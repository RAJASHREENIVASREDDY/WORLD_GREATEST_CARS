CSS Styling
Three ways to style HTML — all demonstrated in this car project
01 — Inline CSS

Inline CSS is written directly inside an HTML element using the style="" attribute. It only affects that single element.

Inline CSS Example
<!-- INLINE CSS: style applied directly on the element -->
<p style="color: red; font-size: 20px; border: 2px solid red; padding: 10px;">
    Ferrari — Rosso Corsa (Racing Red)
</p>

<p style="color: #003087; font-weight: bold; background: #e8f0ff; padding: 10px;">
    BMW — Bavarian Blue Heritage
</p>
Live Output

🇮🇹 Ferrari — Rosso Corsa (Racing Red)

🇩🇪 BMW — Bavarian Blue Heritage

Advantage: Quick, no extra files needed Inline
Disadvantage: Hard to maintain, can't reuse styles Best for: quick one-off overrides
02 — Internal CSS

Internal CSS is written inside a <style> block in the HTML <head>. It applies to the whole page.

Internal CSS — inside <head>
<head>
  <style>
    /* INTERNAL CSS: car brand color themes */
    .ferrari-theme {
      background: #8b0000;
      color: #fff;
      border: 1px solid #cc0000;
    }
    .bmw-theme {
      background: #003087;
      color: #fff;
      border: 1px solid #0066cc;
    }
    .lamborghini-theme {
      background: #4a3c00;
      color: #f5c518;
      border: 1px solid #c9a800;
    }
  </style>
</head>

Advantage: Styles reused across the page Internal
03 — External CSS

External CSS lives in a separate .css file and is linked to the HTML. Used across all pages of this project.

index.html — linking external file
<head>
  <!-- EXTERNAL CSS: link to shared stylesheet -->
  <link rel="stylesheet" href="style.css">
</head>
style.css — external file contents (excerpt)
/* External CSS: applies to every page in the project */
.navbar {
  position: sticky;
  top: 0;
  background: rgba(10,10,10,0.95);
  border-bottom: 1px solid #2a2a2a;
}

.btn {
  font-family: 'Share Tech Mono', monospace;
  padding: 10px 24px;
  border: 1px solid #e63946;
  transition: all 0.2s;
}
